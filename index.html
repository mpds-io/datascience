Dear physicists, chemists, and engineers! We are happy to welcome you here. We know that you nowadays have to process very large amounts of information while research, so we have created MPDS to assist you.

The MPDS contains materials data, manually extracted by our experts from scientific publications. We don't believe this task can be somehow automatized and since 1993 we process scientific articles in physics, chemistry, materials science etc. literally by hands. The result is now in front of you.

§1. What is PAULING FILE?

The PAULING FILE is the relational database for materials scientists, grouping crystallographic data, phase diagrams, and physical properties of inorganic crystalline substances under the same frame. The focus is put on the experimental observations, and the data are processed from the original publications, covering world scientific literature from 1891 to present date. Each individual crystal structure, phase diagram, or physical property database entry originates from a particular publication.

§2. What is MPDS platform?

§2.1. Introduction

The MPDS platform is the online edition of the PAULING FILE materials database. All the data are presented in two ways (online interfaces): browser-based graphical user interface (GUI) and uniform application programming interface (API). Here the browser user interface (GUI) is described.

Search of data at MPDS platform is possible according to 11 criteria: 7 in physics and chemistry (materials classes, physical properties, chemical elements, chemical formulae, space groups, crystal systems, prototypes) and 4 in bibliography (publication author, years, journal, and DOI). There are two search modes: basic and advanced.

In the basic mode different search terms can be typed in a single input field. The most frequently used criteria are supported in this field: materials classes, physical properties, chemical elements, chemical formulae, and crystal systems. In the advanced mode each of 11 search criteria has its own input field. Let us get acquainted with the meaning and proper usage of each search criterion.

§2.2. Materials classes

Various materials classes are collected in this category, ranging from technical terms to physical categories, chemical names, periodic table groups, some isotope names etc. There are also lot of auxiliary terms, e.g. cell-only, cell and atoms, disordered, and non-disordered, applicable to crystalline structures only. §2.3. Physical properties

The supported physical properties are given by the MPDS hierarchy. A search for a high-order property assumes all the subordinate properties. In addition, even more general terms like spin or pressure are supported (the search yields all the physical properties containing this term).

§2.4. Chemical elements

Chemical elements can be typed as full names or symbols. Obviously, chemical elements can be combined arbitrary in search, using spaces, commas, or dashes as separators. By default, equal or greater number of elements is implied, e.g. the results for Cd-O-S will also contain Tl, H, N, K etc. To restrict the elements count, special materials classes unary, binary, ternary, quaternary, or quinary should be added, e.g. like this: Cd-O-S ternary.
